In this codelab you will learn the core concepts for TensorFlow 2.0 and will apply them to solve a classical MRI problem in a few lines of code. We will not focus on neural networks, but rather on the powerful automatic differentiation capabilities of TensorFlow which can be used for a wide range of applications.

What you'll learn

  • What is Tensor and how it differs from a Numpy array
  • How to define a computational graph and compute gradients
  • How to introduce variables, and perform optimization
  • How to apply these techniques to solve an MRI problem

What you'll need

Nothing but a web browser, everything will happen on Google Colaboratory.
